## Description

**Portfolio Administration** is a web-based application for managing portfolio-related data, user authentication, and authorization. It features a modern Administration Panel for easy CRUD operations on portfolio entities such as competences, education, experiences, information, messages, and projects.

This project leverages the power of Bootstrap for responsive design, Django for the backend, and Supabase (PostgreSQL) for database management, offering a robust, scalable, and user-friendly admin experience.

---

## Technologies Used

- **Bootstrap** – Frontend framework for modern, responsive UI
- **Django** – Backend framework for secure, scalable server-side logic
- **Supabase (PostgreSQL)** – Cloud database for reliable data storage
- **CSS** – Custom styles
- **JavaScript** – Interactivity and dynamic UI
- **HTML** – Markup and layout
- **Python** – Backend development with Django

---

## Features

- **Authentication and Authorization**
  - Manage user accounts and groups
- **Admin Panel**
  - Intuitive dashboard for managing all portfolio entities
- **Portfolio Management**
  - Competences: Add or edit professional skills
  - Educations: Manage educational history
  - Experiences: Track work experience
  - Informations: Store personal info
  - Messages: View and respond to communications
  - Projects: Showcase and organize projects
- **Recent Actions**
  - Quick view of latest admin activities
